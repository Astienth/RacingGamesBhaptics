# Project-Cars-Bhaptics

## About the project ?

Project-Cars-Bhaptics is an API based project, allowing you to use bhaptics products with project cars 1 & 2

## Compatibility

Curently :
- X40
- X16
- Arms

## Event Feedback

- Engine
- GearWheel
- Gear
- GForce

next :
- Crash
- missing tyre

## How to install
